1. 서비스 목적

구독청소는 사용자가 직접 입력한 결제내역 텍스트를 바탕으로 반복결제 후보를 정리하는 보조 도구입니다.

2. 제공하지 않는 기능

구독청소는 금융 자문, 결제 취소, 구독 해지 대행, 카드사 조회, 은행 계좌 조회, 본인인증, 신용평가 서비스를 제공하지 않습니다.

3. 분석 결과의 한계

반복결제 후보, 신뢰도, 다음 결제 예상일은 입력된 텍스트를 바탕으로 계산된 참고 정보입니다. 실제 구독 여부와 청구 여부는 카드사, 결제처, 은행 내역에서 직접 확인해야 합니다.

4. 사용자 책임

사용자는 민감정보를 입력하지 않아야 하며, 분석 결과를 바탕으로 한 해지, 유지, 결제 확인 등의 결정은 본인 책임으로 진행합니다.
