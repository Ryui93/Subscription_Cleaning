# 장비 / 오버로드 / 오버클럭 데이터 가이드

## 목적
장비와 오버로드 상태는 같은 니케라도 실제 계정 투자 상태에 따라 추천 우선순위가 달라지는 문제를 보정하기 위한 사용자 입력 데이터다.

## 장비 슬롯
- 머리
- 장갑
- 몸통
- 신발

## 장비 티어
- 없음
- T7
- T8
- T9
- T9 기업
- 오버로드
- 미확인

## 장비 강화
- 슬롯별 강화 레벨 입력
- 추천 점수에는 소폭만 반영
- 정확한 전투력 공식으로 계산하지 않음

## 오버로드
- 슬롯별 오버로드 여부 입력
- 주요 옵션 입력
- 딜러/지원/아레나에 따라 참고 가치가 다름
- 정확한 옵션 효율은 검수 필요

## 오버클럭
- 장비 오버로드와 별도
- 시뮬레이션룸/고난도 조건 참고값
- 생존/유지력 역할 가치에 영향을 줄 수 있는 메모 기반 조건으로 처리

## 주의사항
- 실제 게임 전투력 공식을 복제하지 않음
- 미입력 상태는 보정 0
- 점수 보정은 추천 보조값
- 과도한 보정 금지
