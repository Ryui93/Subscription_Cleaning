# 계정 성장 데이터 가이드

## 목적
계정 성장 데이터는 같은 니케라도 실제 계정 투자 상태에 따라 추천 우선순위가 달라지는 문제를 보정하기 위한 사용자 입력 데이터다.

## 포함 항목
- 호감도
- 싱크로 디바이스
- 리사이클룸
- 택틱 아카데미

## 호감도
- 니케별 입력
- 추천 점수에 소폭 반영
- 육성 추천 사유에 표시

## 싱크로 디바이스
- 계정 전체 레벨 기준
- 직접적인 전투 공식 계산이 아니라 추천 컨텍스트로 사용
- 미입력 시 기본 추천만 제공

## 리사이클룸
- 클래스별/제조사별 투자 상태 입력
- 추천 점수에 소폭 반영
- 과도한 점수 보정 금지

## 택틱 아카데미
- 진행 상황 메모
- 직접적인 점수 계산보다 계정 성장 기록용

## 주의사항
- 실제 게임 전투력 공식을 복제하지 않음
- 정확한 수치가 필요한 경우 추후 검수 데이터 필요
- 현재는 추천 알고리즘용 보조 보정값으로 사용
