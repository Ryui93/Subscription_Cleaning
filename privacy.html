1. 수집하는 정보

v0.1.0 기준 구독청소는 회원가입, 로그인, 서버 저장을 제공하지 않습니다. 사용자가 입력한 결제 알림 텍스트, 분석 결과, 분류 상태, 고객 요청 방명록 내용은 현재 브라우저의 localStorage에 저장됩니다.

2. 서버 전송 여부

입력한 결제내역과 분석 결과는 구독청소 서버로 전송되지 않습니다. 현재 버전에는 외부 데이터베이스, 금융 API, 본인인증 API, 광고 스크립트가 포함되어 있지 않습니다.

3. 보관과 삭제

데이터는 사용자의 브라우저에 남아 있으며, 앱의 전체 데이터 삭제 버튼 또는 브라우저 저장공간 삭제 기능으로 지울 수 있습니다.

4. 입력하지 말아야 할 정보

카드번호 전체, 주민등록번호, 계좌번호 전체, 비밀번호, 인증서 비밀번호, 공동인증서 파일 등 민감정보는 입력하지 마세요.
